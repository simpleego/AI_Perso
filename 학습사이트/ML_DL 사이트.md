# 머신러닝 딥러닝 사이트
> **머신러닝·딥러닝을 시각적으로 체험할 수 있는 대표적인 사이트로는 TensorFlow의 *Neural Network Playground*, Google의 *Teachable Machine*,
> 그리고 다양한 실습 예제를 제공하는 *AI Teading* 같은 플랫폼이 있습니다. 이들은 웹 브라우저에서 바로 실행 가능하며,
> 모델 구조와 학습 과정을 직관적으로 확인할 수 있습니다.**

---

## 🔹 대표적인 체험 사이트

| 사이트 | 특징 | 활용 예시 |
|--------|------|-----------|
| **Neural Network Playground (TensorFlow)** | 브라우저에서 신경망 구조를 직접 조정하며 학습 과정을 시각적으로 확인 가능 | 학습률, 활성화 함수, 레이어 수 변경 시 결과 비교 |
| **Google Teachable Machine** | 이미지·소리·포즈 데이터를 업로드해 간단히 모델을 학습시킬 수 있음 | 웹캠으로 제스처 인식, 음성 명령 분류 |
| **AI Teading** | CNN, RNN, GAN, DQN 등 다양한 딥러닝 알고리즘을 게임·시뮬레이션 형태로 체험 | 자율주행 시뮬레이터, 그림 인식 게임, 얼굴 인식 실습 |
| **RunwayML / Hugging Face Spaces** | 최신 생성형 AI 모델을 웹에서 바로 실행 가능 | 텍스트→이미지 변환, 음성 합성, 챗봇 체험 |

---

## 🔹 학습 체험 방식
- **직관적 시각화**: 데이터 분포, 뉴런 연결, 가중치 변화 등을 색상과 그래프로 확인 가능  
- **실시간 조정**: 학습률, 레이어 수, 규제(regularization) 등을 바꾸며 결과 비교  
- **간단한 데이터 업로드**: 사용자가 직접 이미지·텍스트·음성을 넣어 모델을 학습시키고 결과 확인  
- **게임형 체험**: DQN 기반 게임(스네이크, 자동차 주행 등)을 통해 강화학습 원리 체험

- **입문자**: Neural Network Playground, Teachable Machine → 직관적으로 원리 이해

- **실무자**: Hugging Face Spaces → 최신 모델 직접 실행 및 코드 확인
