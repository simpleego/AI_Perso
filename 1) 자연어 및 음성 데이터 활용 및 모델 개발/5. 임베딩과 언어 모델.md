# 5. 임베딩과 언어 모델

# 1. 임베딩(Embedding)의 핵심 개념

## 1) 정의

* 단어/토큰 → **고정 길이 실수 벡터**
* 목적: 컴퓨터가 **의미를 계산 가능하게 만들기**

## 2) 중요한 성질

* 의미적으로 유사한 단어 → 벡터 공간에서 가까움
* 선형 연산 가능
  예:
  `king - man + woman ≈ queen`

---

# 2. 전통적 표현 방법 (Sparse Representation)

## 2.1 원-핫 인코딩 (One-hot Encoding)

### 개념

* 단어 수 = 차원 수
* 하나만 1, 나머지 0 → 매우 희소(sparse)

### 문제점

* 의미 정보 없음
* 차원 폭발

### 코드

```python
from sklearn.preprocessing import OneHotEncoder
import numpy as np

words = np.array(["cat", "dog", "fish", "cat"]).reshape(-1, 1)

encoder = OneHotEncoder(sparse_output=False)
one_hot = encoder.fit_transform(words)

print(one_hot)
print(encoder.categories_)
```

---

## 2.2 TF-IDF

### 개념

* TF (Term Frequency): 단어 빈도
* IDF (Inverse Document Frequency): 희귀성

→ 중요 단어에 높은 값

### 수식

* TF-IDF = TF × log(N / DF)

---

### 코드

```python
from sklearn.feature_extraction.text import TfidfVectorizer

corpus = [
    "I love NLP",
    "I love machine learning",
    "NLP is fun"
]

vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(corpus)

print(vectorizer.get_feature_names_out())
print(X.toarray())
```

---

## 2.3 PMI (Pointwise Mutual Information)

### 개념

* 두 단어가 **함께 등장할 확률**
* 독립일 때보다 얼마나 더 자주 등장하는가

### 수식

[
PMI(x,y) = \log \frac{P(x,y)}{P(x)P(y)}
]

---

### 간단 코드

```python
import numpy as np

# co-occurrence matrix 예시
co_matrix = np.array([
    [10, 2],
    [2, 8]
])

total = np.sum(co_matrix)

p_xy = co_matrix / total
p_x = np.sum(p_xy, axis=1)
p_y = np.sum(p_xy, axis=0)

pmi = np.log(p_xy / (p_x[:, None] * p_y[None, :]))
print(pmi)
```

# 3. 예측 기반 임베딩 (Prediction-based)

## 3.1 Word2Vec

### 핵심 아이디어

* 주변 단어를 보고 중심 단어를 예측 (CBOW)
* 중심 단어로 주변 단어를 예측 (Skip-gram)

---

### 코드 (gensim)

```python
from gensim.models import Word2Vec

sentences = [
    ["i", "love", "nlp"],
    ["i", "love", "machine", "learning"],
    ["nlp", "is", "fun"]
]

model = Word2Vec(sentences, vector_size=50, window=2, min_count=1)

print(model.wv["nlp"])
print(model.wv.most_similar("nlp"))
```

---

## 3.2 FastText

### 특징

* 단어를 **subword (n-gram)** 단위로 분해
* OOV(없는 단어) 처리 가능

---

### 코드

```python
from gensim.models import FastText

model = FastText(sentences, vector_size=50, window=2, min_count=1)

print(model.wv["nlp"])
print(model.wv["nlppp"])  # OOV도 벡터 생성 가능
```

# 4. 문맥 기반 임베딩 (Contextual Embedding)

## 4.1 ELMo

### 특징

* LSTM 기반
* 문맥에 따라 단어 의미 변화

예:

* bank (river bank vs bank account)

---

## 4.2 BERT

### 특징

* Transformer 기반
* 양방향 문맥 이해
* MLM (Masked Language Model)

---

### 코드 (HuggingFace Transformers)

```python
from transformers import BertTokenizer, BertModel
import torch

tokenizer = BertTokenizer.from_pretrained("bert-base-uncased")
model = BertModel.from_pretrained("bert-base-uncased")

sentence = "I love NLP"
inputs = tokenizer(sentence, return_tensors="pt")

outputs = model(**inputs)

# 마지막 hidden state
embeddings = outputs.last_hidden_state
print(embeddings.shape)
```

# 5. 전체 흐름 정리

| 방법       | 특징        | 한계         |
| -------- | --------- | ---------- |
| One-hot  | 단순        | 의미 없음      |
| TF-IDF   | 중요도 반영    | 문맥 없음      |
| PMI      | 통계적 의미    | sparse     |
| Word2Vec | 의미 학습     | 문맥 제한      |
| FastText | OOV 해결    | subword 의존 |
| ELMo     | 문맥 반영     | 느림         |
| BERT     | 강력한 문맥 이해 | 연산량 큼      |
